Tips & Best Practices

Get the most out of Exam Grader

📝 Marking Guide Tips
  • Include clear question numbers and point values
  • Provide detailed expected answers or rubrics
  • Use consistent formatting throughout
  • Include partial credit guidelines when applicable
📄 Submission Tips
  • Ensure good image quality for scanned documents
  • Use high contrast (dark text on light background)
  • Avoid shadows and glare in photos
  • Keep file sizes reasonable for faster processing
⚡ Performance Tips
  • Use "Fast" model for quick grading of simple questions
  • Use "Accurate" model for complex or subjective content
  • Process smaller batches for faster results
  • Clear cache regularly to free up storage space
✅ Quality Assurance
  • Always review AI-generated grades before finalizing
  • Check OCR-extracted text for accuracy
  • Use the detailed feedback to understand grading logic
  • Keep backup copies of original submissions

Troubleshooting

Common issues and solutions

Upload Failed

Problem: File upload returns an error

Solutions:

  • Check file size (max 20MB)
  • Verify file format is supported
  • Try refreshing the page
  • Clear browser cache
OCR Not Working

Problem: Text extraction from images fails

Solutions:

  • Ensure good image quality
  • Check for proper lighting
  • Try different image format
  • Manually type text if needed
Slow Processing

Problem: AI grading takes too long

Solutions:

  • Switch to "Fast" model in Settings
  • Process smaller batches
  • Clear cache to free memory
  • Check internet connection
Grading Errors

Problem: AI produces incorrect grades

Solutions:

  • Review marking guide clarity
  • Use "Accurate" model for complex questions
  • Manually review and adjust grades
  • Provide more detailed rubrics
Still need help? If you're experiencing persistent issues, check the system status in Settings or contact your system administrator.
